# 🧠 DIKW Memory Plugin

> 基于 DIKW 模型的智能记忆管理系统 for Hermes Agent

[![Version](https://img.shields.io/badge/version-v7.7.5-blue)](https://github.com/Zhao961215/Dikwl-memory-v7.7.5)
[![License](https://img.shields.io/badge/license-MIT-green)](LICENSE)
[![Platform](https://img.shields.io/badge/platform-Hermes%20Agent-orange)](https://github.com/Zhao961215/Dikwl-memory-v7.7.5)

## ✨ 特性

| 功能 | 描述 |
|------|------|
| 🧠 `memory_learn` | 记忆学习 - 沉淀原则/教训，智能分流 I/D/K/W 类 |
| 🔍 `memory_recall` | 记忆召回 - 11步防漏检索，引擎优先+文件兜底 |
| 📈 `memory_evolve` | 记忆进化 - 纠正旧认知，升级方法论 |
| ⚙️ `memory_admin` | 记忆管理 - SNR监控/清理/健康检查 |

## 📦 安装

```bash
# 克隆仓库
git clone https://github.com/Zhao961215/Dikwl-memory-v7.7.5.git

# 复制文件到插件目录
cp -r *.py *.yaml ~/.hermes/plugins/dikw/

# 重启 Hermes Gateway
hermes restart
```
